# 🔬 Guialab

O **Guialab** é um sistema web responsivo e minimalista desenvolvido para facilitar o dia a dia de profissionais de saúde e laboratórios. O objetivo principal é guiar o usuário na identificação rápida dos **códigos corretos** e dos **tubos de coleta específicos** para cada exame solicitado, minimizando erros na fase pré-analítica.

---

## 🚀 Funcionalidades

* 🔍 **Busca em tempo real:** Pesquisa instantânea por nome do exame ou código.
* 📋 **Tabela Inteligente:** Exibição clara e organizada dos dados essenciais (Exame, Código e Tubo).
* 🎨 **Identificação Visual:** Cores dos tubos padronizadas via CSS para facilitar a associação visual (ex: Roxo para EDTA, Azul para Citrato).
* 📱 **Layout Responsivo:** Totalmente adaptado para uso em computadores, tablets ou smartphones.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** - Estruturação semântica do sistema.
* **CSS3** - Estilização customizada, responsividade e identidade visual (Verde Guialab).
* **JavaScript / jQuery** - Manipulação dinâmica da tabela e lógica de busca.
